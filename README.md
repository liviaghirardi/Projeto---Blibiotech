# Projeto---Blibiotech
📌 Sobre o Projeto

O projeto consiste em uma página inicial moderna com:

- Cabeçalho com logo e menu de navegação

- Botão de login para membros

- Seção principal (Hero) com imagem de fundo

- Frase de destaque incentivando o acesso ao conhecimento

A proposta é apresentar uma biblioteca tecnológica com visual limpo, organizado e responsivo.

🛠️ Tecnologias Utilizadas
- HTML5

- CSS3

- VS CODE 

Flexbox para alinhamento e organização dos elementos
